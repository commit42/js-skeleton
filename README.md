# js-skeleton

A starter kit to build basic js lib

## Contributing

## With docker (recommended)

### Install in local env

```bash
yarn && yarn c42 install
```

### Run tests

```bash
yarn c42 node:test
```

## Without docker

require node 12 (node 10 is probably compatible)

### Install in local env

```bash
yarn

# or

npm install
```

### Run tests

```bash
yarn test

# or

npm run test
```
